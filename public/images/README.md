# Where to put photos

Drop real photos into these folders using these exact filenames (or tell me
the filenames you used and I'll wire them in). This keeps the code changes
simple — once files exist with these names, I flip a couple of lines and
they show up on the live site.

## /images/doctor/
- headshot.jpg        — main professional headshot (used in hero sections, About page)
- consultation-room.jpg — Dr. Gowda in the consultation room (About page)
- profile-square.jpg   — square crop, used for the small circle/card versions

## /images/hospital/
- exterior.jpg         — HIMAS Hospital building/entrance
- reception.jpg        — reception/waiting area
- ot.jpg                — operation theatre (if allowed to photograph)

## /images/patients/
Only include photos with explicit patient consent.
- patient-1.jpg, patient-2.jpg, etc. — for Patient Stories page

## /images/achievements/
- award-1.jpg, award-2.jpg, etc. — photos of actual awards/certificates/ceremonies

## /images/blog/
- post-1-cover.jpg, post-2-cover.jpg — cover images for blog posts

## Guidelines
- JPG or PNG, ideally under 500KB each (I can compress if needed)
- Landscape orientation works best for hero/cover images
- Square (1:1) works best for headshots/avatars
